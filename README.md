# Cheap School Data

This project aims to provide a structured solution for managing school data efficiently.

## Project Structure

### Backend
- `backend/`
  - `src/`  (Contains source code for the backend)
  - `tests/`  (Unit tests for the backend)
  - `README.md`  (Documentation for backend setup)
  - `requirements.txt`  (Python dependencies)
  - `Dockerfile`  (Containerization file)

### Frontend
- `frontend/`
  - `src/`  (Contains source code for the frontend)
  - `public/`  (Static files)
  - `tests/`  (Unit tests for the frontend)
  - `README.md`  (Documentation for frontend setup)
  - `package.json`  (Javascript dependencies)
  - `Dockerfile`  (Containerization file)

### Documentation
- `docs/`
  - `architecture.md`  (High-level architecture overview)
  - `api.md`  (API documentation)
  - `setup.md`  (Installation and setup instructions)
  - `CONTRIBUTING.md`  (Guidelines for contributing to the project)
